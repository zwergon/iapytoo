# Inference using MlServer

In this directory, we illustrate how inference can be made using a model generated by Iapytoo.
We assume that we ran `examples/wgan_train.py`, this will produce a model that is saved by mlflow.
This model is then stored in the `models` directory.

### MLServer
There is a python launch of mlserver `debug_mlserver.py` that may be used for debug purposes.
Once lauched, it exposed two ports on `localhost` 
1. One for the http endpoint (8080) using `v2` protocol
1. One for the grpc endpoint (8081) 


### Inference
* `test_infer.py` To make this inference using the `http` endpoint nothing else is required than python (and requests module).
* `test_infer_grpc.py`For the inference using `grpc` endpoint, python and `tritonclient[grpc]` are required.
* `test_infer_codec.py` There is also a code that uses the `MlInputCodec` from `Iapytoo` to make a http request.
